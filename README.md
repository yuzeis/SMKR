# RocoMITMServer Ver2.3 Untergangsgeweiht

RocoMITMServer 是面向洛克王国流量的本地研究工具，提供 MITM 代理、Web 控制台、RFN 运行时、MCP 接口、协议结构观察和性能诊断能力。

本项目仅用于安全研究、逆向学习、个人协议分析、教学示例和互操作性验证。使用者应自行确认操作符合适用法律法规、服务条款、用户协议、EULA 和第三方权利要求。

## 快速启动

源码目录内运行:

```powershell
$env:PYTHONPATH = "src"
python -m roco_mitm web
```

Windows 双击启动:

```powershell
.\run.bat
```

PowerShell 脚本:

```powershell
.\scripts\run_web.ps1
.\scripts\run_divert.ps1
```

安装为本地开发包:

```powershell
python -m pip install -e ".[dev]"
roco-mitm web
```

```powershell
$env:ROCO_MITM_ROOT = "<project-root>"

$env:ROCO_MITM_RUNTIME_DIR = "<runtime-dir>"
```

## 主要能力

- 本地 MITM 代理和 Web 控制台。
- RFN 脚本运行时和在线数据包观察。
- MCP 工具接口，便于外部自动化调用。
- Opcode、消息结构、模板和配置文件的本地加载。
- 代理性能统计、详细事件采样和 JSONL 导出。
- 可配置的 C2S 批量发送、S2C 观察、自动解码和 RFN 包投递策略。
