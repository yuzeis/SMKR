RocoMITMServer 是面向洛克王国流量的研究工具。

本项目仅用于安全研究、逆向学习、个人协议分析和互操作性验证。使用者应自行确保操作符合适用法律法规、服务条款、用户协议、EULA 和第三方权利要求。

## 快速启动

源码目录内运行:

```powershell
$env:PYTHONPATH = "src"
python -m roco_mitm web
```

Windows 双击启动:

```powershell
.\run.bat
```

PowerShell 脚本:

```powershell
.\scripts\run_web.ps1
.\scripts\run_divert.ps1
```

安装为本地开发包:

```powershell
python -m pip install -e ".[dev]"
roco-mitm web
```

路径环境变量:

```powershell
$env:ROCO_MITM_ROOT = "<project-root>"
$env:ROCO_MITM_CONFIG_DIR = "<config-dir>"
$env:ROCO_MITM_RUNTIME_DIR = "<runtime-dir>"
```
